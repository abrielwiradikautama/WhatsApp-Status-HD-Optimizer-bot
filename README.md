# WhatsApp-Status-HD-Optimizer-bot

Windows batch script to search and optimize WhatsApp status media (videos & images) using FFmpeg.

## Features
- Deep system-wide search for media files (MP4, MKV, MOV, JPG, PNG, HEIC)
- Automatic FFmpeg detection or download if missing
- Optimize videos to HD (1080x1920) with x264 codec
- Optimize images up to 2560px while preserving aspect ratio
- Simple command-line interface

## Requirements
- Windows 10/11
- FFmpeg (automatic download included in script)
- Batch script execution enabled

## Usage
1. Place `whatsapp HD.bat` in any folder
2. Run the batch script
3. Enter keyword or file name to search
4. Select the media from the list
5. Optimized files will be saved in `Optimized_Status` folder
